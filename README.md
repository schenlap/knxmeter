# knxmeter
knxmeter is a device for reading data from a smart meter (as first with AMIS smart meter from austrian netzOÃ–) and sending it to the knx bus. The development of knxmeter is done with open source tools only.

Hardware
--------
A STM32 microcontroller is used, connetion with KNX bus is done with Siemens TPUART2.
Hardware schematic and pcb is created with the power of KiCad.
